# BulkWave Pro 🚀
### Smart Bulk WhatsApp Automation

A professional desktop application for bulk WhatsApp messaging, built with Python, CustomTkinter, and Selenium.

---

## Screenshots

> Launch the app with `python main.py` to see the premium dark-mode SaaS-style interface.

---

## Features

| Feature | Details |
|---|---|
| **Bulk Messaging** | Send personalised messages to hundreds of contacts |
| **Excel / CSV Upload** | Drag-and-drop support, auto-detects phone/name columns |
| **Start From Row** | Resume interrupted campaigns from any row |
| **Message Placeholders** | `{name}` and `{number}` personalisation |
| **Image Attachments** | Optional JPG/PNG attachment per message |
| **Delay Control** | Configurable random delay (min/max seconds) |
| **Live Logs** | Real-time per-message status in the UI |
| **Progress Dashboard** | Progress bar, ETA, sent/failed/remaining counters |
| **WhatsApp Session** | Chrome profile persistence — scan QR code once |
| **Pause / Resume / Stop** | Full campaign control |
| **History** | All past campaigns with export of failed contacts |
| **Templates** | Save and load message templates |
| **Settings** | Country code, delays, retry settings, and more |
| **Export Failed** | Download failed contacts as Excel for re-campaign |

## Quick Start

### 1. Clone / Download

```
git clone <repo-url>
cd "Smart Bulk WhatsApp Automation"
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Generate sample contacts (optional)

```bash
python create_sample.py
```

### 4. Launch the app

```bash
python main.py
```

## Using the App

### Step-by-step workflow

1. **Open WhatsApp Web** — Click the green "Open WhatsApp Web" button in the Bulk Sender page. A Chrome window will appear with the WhatsApp Web login page.

2. **Scan QR Code** — Use your phone's WhatsApp app to scan the QR code. The session is saved locally so you only need to do this once.

3. **Upload Contacts** — Click the upload zone and select your `.xlsx`, `.xls`, or `.csv` file. The app auto-detects the phone number column.

4. **Configure** — Select the Phone Column, (optional) Name Column, and set the Start Row if resuming.

5. **Compose Message** — Type your message. Use `{name}` and `{number}` for personalisation.

6. **Attach Image** (optional) — Click "Browse Image" to add a JPG/PNG attachment.

7. **Set Delays** — Configure minimum and maximum delay between messages (recommended: 3–7 seconds minimum).

8. **Start Sending** — Click "▶ Start Sending". Monitor progress in real time.

## Excel File Format

Your Excel/CSV file should have at least one column with phone numbers in **international format** (e.g. `919876543210` for India). A name column is optional.

| Name | Phone | City |
|---|---|---|
| John Doe | 919876543210 | Mumbai |
| Jane Smith | 919812345678 | Delhi |

> A ready-to-use sample is generated by `python create_sample.py` → `data/sample_contacts.xlsx`.

## Building the Windows EXE

### Option 1 — Run the batch file

```
build.bat
```

### Option 2 — Manual PyInstaller command

```bash
pip install pyinstaller
pyinstaller --noconfirm --onefile --windowed --name "BulkWave Pro" --collect-all customtkinter main.py
```

The EXE will be placed in `dist/BulkWave Pro.exe`.

> **Note:** The first run after building may take 10–15 seconds as PyInstaller extracts resources.

## Configuration

All settings are saved to `data/BulkWave.db` (SQLite) and can be changed on the **Settings** page.

| Setting | Default | Description |
|---|---|---|
| Default Country Code | `91` | Prepended to 10-digit numbers |
| Min Delay (s) | `3` | Minimum pause between messages |
| Max Delay (s) | `7` | Maximum pause between messages |
| Auto-Retry | `true` | Retry failed messages automatically |
| Max Retries | `2` | Retry attempts per contact |

## ⚠ Important — Use Responsibly

> Sending unsolicited bulk messages may violate [WhatsApp's Terms of Service](https://www.whatsapp.com/legal/terms-of-service) and result in your account being temporarily or permanently restricted.
>
> - Always obtain explicit consent before messaging contacts.
> - Use realistic delays (3+ seconds between messages).
> - Avoid sending to very large lists in a single session.
> - This tool is intended for legitimate business communication with opted-in recipients.
